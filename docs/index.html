Built Different

Every component built from scratch in Rust. No Chromium, no WebKit, no Gecko.

🦀

Pure Rust

57,000+ lines of safe Rust code. Zero unsafe blocks. Memory safety guaranteed at compile time. Edition 2024.

🎨

170+ CSS Properties

Flexbox, Grid, table layout, transforms, animations, transitions, variables, gradients, shadows, and more.

⚡

JavaScript Engine

QuickJS runtime with full DOM API, Canvas 2D, Shadow DOM, WebSocket, XHR, and 50+ Web API stubs.

🚀

GPU Rendering

Vello + wgpu for GPU-accelerated rendering. Automatic fallback to FreeType software renderer.

🌐

Full Networking

HTTPS/TLS, cookies, HTTP cache, WebSocket, HSTS, CORS, CSP. Gzip decompression and redirect following.

💻

Complete Browser Shell

Multi-tab, DevTools, bookmarks, history, find in page, zoom, clipboard, smooth scrolling, keyboard shortcuts.

Micro-Kernel Architecture

The core knows nothing about web standards. All functionality is provided by dynamically-loaded mods.

nova-browser/ # Workspace root ├── nova-core/ # Micro-kernel: loads mods, routes IPC ├── nova-mod-api/ # Trait definitions + TypedData ├── nova-gpu/ # Vello + wgpu compositor ├── nova-pipeline/ # URL → pixels orchestration ├── nova-shell/ # iced UI: tabs, URL bar, DevTools └── mods/ ├── mod-network/ # HTTP, cookies, cache, WebSocket ├── mod-html-parser/ # html5ever ├── mod-css-engine/ # CSS cascade, variables, animations ├── mod-layout/ # Taffy flex/grid + table + IFC ├── mod-painter/ # Layout → RenderOps ├── mod-js/ # QuickJS, DOM API, Canvas 2D └── mod-image/ # PNG, JPEG, WebP, GIF, SVG
